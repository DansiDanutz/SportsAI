7. Arbitrage Mathematics

2-Way Formula

Arbitrage % = (1/Odds_A) + (1/Odds_B) - 1

If result < 0: GUARANTEED PROFIT

Example

Odds_A = 2.10, Odds_B = 1.95

Arbitrage % = (1/2.10) + (1/1.95) - 1
           = 0.476 + 0.513 - 1
           = -0.011 = -1.1%

PROFIT: 1.1% guaranteed
